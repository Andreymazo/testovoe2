устанавливаете проект, окружение + pip install -r requirements.txt.
Подключаем базу (миграции).
Командными файлами заполняем базу.

Суть задания - вывод на одном темплейте несколько сущностей, помимо основной, еще детей родительской сущности.

Посмотрел, как исполнено это задание у других, решил, сделать с библиотекой MPTT.
В папке: menu/management/commands/ лежат командные файлы, в конце каждого цифра - его очередь запуска. С ними удобно удалять целиком базу и менять что-либо.
В проекте 3 сущности: Menu, Menu(child), MenuItem. По-сути Меню - родитель к двум другим, но хотелось попробовать вывести на один шаблон три сущности. Темплейты по адресу menu/templates. В файле menu.html сначала показывает дочерние сущности (библиотека mptt, код с tutorial, у нас это будут два родителя Rock, Blues и ребенок - Classic music). Дальше идет повторение вывода циклом родителя и его детей, но уже MenuItem, то есть библиотека мптт этих детей не видит, далее в конце вывод родителей и инфы по ним, например, а есть ли дети (как я узнал, что библиотека видит).
Админка: реализована двумя способами, вывод чуть отличается, сейчас раскоментирован вариант, когда сразу видны родитель и дети по мптт. Второй вариант с Табуляринлайном тоже работает, нагляднее при редактировании родителя с детьми, но нет общей наглядности. 3ий вариант админки у меня пока не заработал.
